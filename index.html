Create Google Sheet
Go to sheets.google.com → New blank sheet → Name it TaskBridge
2
Open Apps Script
In the Sheet: Extensions → Apps Script → Delete all code → Paste the TaskBridge_Backend.gs file → Save
3
Deploy as Web App
Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy → Authorize → Copy the URL
4
Add Your Team
After connecting, go to Team tab → Add all 27 members with their PINs. The app auto-populates the Users sheet.
